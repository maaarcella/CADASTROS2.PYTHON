# 👥 Sistema de Gerenciamento de Usuários em Python

## Descrição

Este projeto consiste em um sistema de gerenciamento de usuários desenvolvido em Python para execução via terminal. A aplicação permite cadastrar, listar, alterar e excluir usuários de forma simples e organizada.

Os dados são armazenados em listas durante a execução do programa, possibilitando o gerenciamento de informações como nome, e-mail e telefone. O sistema foi criado com fins educacionais para praticar conceitos fundamentais da programação.

## Funcionalidades

* Cadastro de usuários;
* Listagem de usuários cadastrados;
* Alteração de nome, e-mail ou telefone;
* Exclusão de usuários;
* Menu interativo no terminal;
* Validação básica de opções;
* Interface simples e intuitiva.

## Tecnologias Utilizadas

* Python 3
* Biblioteca padrão `os`

## Objetivo do Projeto

O objetivo deste projeto é aplicar conhecimentos de programação em Python por meio do desenvolvimento de um sistema CRUD (Create, Read, Update e Delete), permitindo realizar as principais operações de gerenciamento de dados.

Durante o desenvolvimento foram praticados conceitos essenciais da linguagem, como estruturas condicionais, laços de repetição, listas, funções de manipulação de dados e interação com o usuário.

## Como Funciona

Ao iniciar o programa, o usuário tem acesso a um menu com as seguintes opções:

1. Cadastrar usuário;
2. Listar usuários;
3. Excluir usuário;
4. Sair do sistema;
5. Alterar cadastro.

Cada usuário possui as seguintes informações:

* Nome completo;
* E-mail;
* Telefone.

As operações podem ser realizadas quantas vezes forem necessárias até que o usuário escolha encerrar o sistema.

## Conceitos Aplicados

* Estruturas condicionais (`if`, `elif`, `else`);
* Estruturas de repetição (`while`);
* Manipulação de listas;
* Funções `enumerate()` e `zip()`;
* Entrada e saída de dados;
* Organização de menus interativos;
* Operações CRUD.
